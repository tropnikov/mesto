# Проект 3: Место

## Обзор

Третий проект на курсе Веб-разработчика в Яндекс.Практикуме.
Вёрстка адаптивной страницы из макета в фигме, реализация нескольких попапов на JS: редактирование имени и био пользователя, добавления нового места в сетку карточек и открытие фотографии в большом размере, валидация форм с помощью JS, закрытие попапов по нажатию Esc. Стек:

- БЭМ
- Флексбоксы
- Гриды
- Небольшие анимации по ховеру
- JS
