# Проект 3: Место

## Обзор

Третий проект на курсе Веб-разработчика в Яндекс.Практикуме.
Вёрстка адаптивной страницы из макета в фигме и реализация попапа на JS, который редактирует имя и био пользователя. Стек:

- БЭМ
- Флексбоксы
- Гриды
- Небольшие анимации по ховеру
- JS
