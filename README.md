# Проект 4: Место

## Обзор

Четвертый проект на курсе Веб-разработчика в Яндекс.Практикуме.

- Вёрстка адаптивной страницы из макета в Фигме
- Создание карточки как экземпляра класса Card
- Реализация нескольких попапов на JS:  

  - редактирование имени и био пользователя
  - добавления нового места в сетку карточек
  - открытие фотографии в большом размере.  

- Валидация форм с помощью JS
- Закрытие попапов по нажатию Esc.
  
## Стек

- БЭМ
- Флексбоксы
- Гриды
- Небольшие анимации по ховеру
- JS
- ООП
